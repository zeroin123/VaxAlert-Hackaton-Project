# VaxAlert: Vaccine Stockout Alert System for Ethiopia

A facility-level vaccine stockout forecasting and alert system for Ethiopia's
Expanded Programme on Immunization (EPI). Forecasts days-to-stockout for 7
EPHI infant-schedule vaccines across 30 health facilities, fires tiered
alerts before stockouts occur, and surfaces cascading failures from Health
Centers to their satellite Health Posts.

## What this project does

- Predicts weekly closing-stock levels 8 weeks ahead per (facility, antigen)
- Triggers operational alerts (critical / warning / ok) based on predicted
  days-to-stockout vs. lead time + safety buffer
- Detects HC-HP cascade failures: when a Health Center runs out of stock,
  its satellite Health Posts cannot collect supply
- Quantifies model performance with MAE, MAPE, interval coverage, and an
  operational stockout-detection-rate metric
- Exposes a Streamlit dashboard with 4 views for national, facility,
  cascade, and model-performance perspectives

## Setup

This project runs on **Python 3.11**.

```bash
pip install -r requirements.txt
```

## Running

The pipeline is: regenerate synthetic data, run cross-validation to train
the stacking meta-model, generate production forecasts, then launch the
dashboard.

```bash
# Optional: regenerate the synthetic database (already shipped pre-built)
python vaxalert_sdg.py

# Train base models + global stacking meta-learner via walk-forward CV
python evaluation/walk_forward_cv.py

# Generate 8-week forecasts for all 210 series and persist feature importance
python forecast/generate_forecasts.py

# Launch dashboard
streamlit run dashboard/app.py
```

For a quick smoke test on 3 series (~1 minute):

```bash
python evaluation/walk_forward_cv.py --sample 3
python forecast/generate_forecasts.py
streamlit run dashboard/app.py
```

The dashboard runs at `http://localhost:8501`.

## Data

The dataset is fully synthetic, generated by `vaxalert_sdg.py` from a fixed
seed (42). The simulation covers **7 years** (364 weeks) of weekly stock
movements for **30 facilities** across **7 antigens** = 210 time series. It
is calibrated against:

- WHO/WUENIC national coverage estimates (2024)
- 2019 Ethiopian Mini Demographic and Health Survey (EMDHS) regional parameters
- Documented Ethiopian EPI stockout rates (Wendrad et al. 2023, Tilmun et al. 2022)
- Real Ethiopian conflict regions (Tigray 2020-2022, Amhara 2023-2024)
- Facility names sampled from the public Ethiopian Ministry of Health facility registry

See `ethics.md` for full data documentation and `PRESENTATION.md` for the
full methodology walk-through.

## Models in the ensemble

- **Naive (last value)** - copy last observed week's stock forward
- **Naive (seasonal)** - copy stock from same week 52 weeks ago
- **Holt-Winters** - exponential smoothing with additive trend + annual seasonality
- **Prophet** - linear trend + Fourier annual seasonality + named events
  (measles SIA, polio SNID, kiremt rainy season, pandemic, regional conflicts)
- **XGBoost** - direct multi-step gradient boosting with lag features
  (1, 2, 4, 8, 12, 26, 52 weeks), rolling stats, calendar encodings, and
  exogenous shock + cascade features (HC stock, recent stockout, lead-time variability)
- **Stacked Ensemble** - a single global XGBoost meta-learner trained on
  pooled out-of-fold base-model predictions across all 210 series with
  conformal prediction intervals

## Dashboard views

1. **National Overview** - 5 KPIs (critical alerts, DTP3 dropout, children
   missed, wastage rate, urgency score), facility map (centered, full-width),
   stacked critical/warning alert tables. Forecast Horizon slider scrubs
   week-by-week through the 8-week horizon.
2. **Facility Drill-Down** - stock history + ensemble forecast with 80%
   prediction interval, top-5 feature importance for the selected facility,
   delivery timeline, session performance.
3. **Cascade View** - HC-HP network diagram, counterfactual replay
   ("if HC was resupplied N weeks earlier, X additional children would
   have been vaccinated"), cascade impact table, cascade-affected weeks heatmap.
4. **Model Performance** - per-model summary (MAE, MAPE, interval coverage,
   stockout detection rate, false alert rate), CV summary across folds,
   MAE × access-tier bar chart, stockout-detection breakdown, feature ×
   access-tier importance heatmap, individual series explorer.

## Key results (final test, weeks 340-363)

| Model | MAE | MAPE | Interval Cov. | Stockout Detection |
|-------|-----|------|---------------|--------------------|
| Stacked Ensemble | **10.81** | 37.6% | 67% | **49.0%** |
| Prophet | 10.61 | 43.7% | 76% | 48.2% |
| Holt-Winters | 11.52 | 48.3% | 70% | 42.5% |
| Naive (Seasonal) | 12.62 | 58.3% | 64% | 43.3% |
| Naive (Last Value) | 13.19 | 60.3% | 65% | 40.9% |
| XGBoost | 17.65 | 59.3% | 56% | 35.2% |

The stacked ensemble achieves the lowest MAE and highest stockout detection
rate, validating the value of model stacking over single-model approaches.

## Ethics

See [ethics.md](ethics.md). Synthetic data only - no real patient records
or individual identifiers. Calibrated against published Ethiopian health
system inequities (pastoral and remote facilities have higher stockout
rates) which the dashboard surfaces explicitly to direct resources to
underserved facilities.
